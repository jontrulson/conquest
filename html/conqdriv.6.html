NAME

conqdriv - Drive a Conquest universe.  

SYNOPSIS

conqdriv [-f]  

DESCRIPTION

conqdriv is a process that is automatically started and stopped by the conquest server conquestd as needed. It is used to drive the universe, and should not be started by ordinary users. Management of the conqdriv process is automatically handled by conquestd as needed.

When run with the -f option, conqdriv will cause any currently running conqdriv process to terminate, and then take control of the universe directly. This option is only of use to developers for debugging purposes.  

OPTIONS

-f
Force any other conqdriv process to terminate, and assume direct control of the universe.
